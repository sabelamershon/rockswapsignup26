One-time setup

This app saves sign-ups online using a free service called JSONBin. Takes 2 minutes — only the organizer needs to do this once.

  1. Go to jsonbin.io and click Sign Up Free
  2. After signing in, click CREATE BIN in the left sidebar
  3. In the editor, paste exactly this: {"volunteers":[],"colorIdx":0} then click CREATE
  4. Copy the BIN ID shown at the top (looks like 6123abc...)
  5. Go to API Keys in the left sidebar, create a key, and copy it
